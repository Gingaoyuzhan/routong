# 肉痛 (Routong)

一款用"社死"驱动自律的 iOS 应用。

## 功能特性

- **契约系统** - 创建自律契约，设置目标、押金和截止日期
- **多种验证方式** - 支持拍照验证、位置打卡、运动轨迹
- **社死惩罚** - 失败后自动发送短信给指定对象（前任、死对头、暗恋对象等）
- **钱包系统** - 支持苹果内购充值

## 技术栈

- SwiftUI
- StoreKit 2 (苹果内购)
- MVVM 架构

## 项目结构

```
routong/
├── Core/           # 核心模块（网络、配置）
├── Features/       # 功能模块
│   ├── Auth/       # 登录认证
│   ├── Contract/   # 契约管理
│   ├── Payment/    # 支付钱包
│   └── Verification/ # 验证系统
└── Shared/         # 共享组件
```

## 环境要求

- iOS 15.0+
- Xcode 14.0+

## 开始使用

1. 克隆项目
2. 用 Xcode 打开 `routong.xcodeproj`
3. 运行项目
